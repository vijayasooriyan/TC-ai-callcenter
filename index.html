⚙️ Setup Guide
Complete steps to get TC-EUSL Vapi voice calls working
1
How Vapi Works
Caller → Vapi Phone Number → Vapi STT: Deepgram transcribes speech → POST to /vapi/llm ← YOUR server → Groq AI answers from TC-EUSL knowledge → Response back to Vapi → Vapi TTS speaks answer to caller → All events → POST to /vapi/webhook → Live Monitor updates in real-time
2
Install & Configure
pip install flask flask-cors groq requests python-dotenv
VariableWhere to get itRequired
VAPI_API_KEYdashboard.vapi.ai → Account → API Keys✓
GROQ_API_KEYconsole.groq.com → API Keys (free)✓
GROQ_MODELopenai/gpt-oss-120b✓
BASE_URLYour ngrok https URL✓
VAPI_PHONE_NUMBER_IDFrom Vapi dashboard → Phone NumbersFor outbound
3
Run & Expose
python app.py # New terminal: ngrok http --domain=unfriendliest-julianne-barbless.ngrok-free.dev 5000 # Update BASE_URL in .env → restart app.py
4
Create Vapi Assistant
Go to the 🎙️ Vapi Setup tab → click Create Assistant.
This automatically configures the assistant with:
• Your custom LLM URL (…)
• TC-EUSL knowledge base in the system prompt
• Multilingual Deepgram STT (auto-detect English/Sinhala/Tamil)
• Polly TTS voice
5
Link Phone Number in Vapi Dashboard
1. Go to dashboard.vapi.ai
2. Phone Numbers → Buy a number or use existing
3. Click the phone number → set Assistant to "TC-EUSL AI Receptionist"
4. Set Server URL to: …
5. Save → Call the number from your phone!
6
Watch Live Monitor
Open the 📡 Live Monitor tab. When a call comes in you'll see every step: caller speech → AI answer → spoken response.
